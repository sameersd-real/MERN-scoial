# MERN Social

A social media web application built with the MERN stack.

## Features

* User signup and login
* User profiles
* React frontend
* Express.js backend

## Installation

### Clone the repository

```bash
git clone https://github.com/sameersd-real/MERN-scoial.git
cd MERN-scoial
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
MONGO_URI=<mongodb url>
PORT=<port if needed>
JWT_SECRET=<some random string>
```

Start the backend:

```bash
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Tech Stack

* React
* Node.js
* Express.js
* MongoDB
* Mongoose
* MERN (basically)
